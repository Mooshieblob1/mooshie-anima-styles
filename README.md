# Anima Style Gallery

A browsable gallery of artist style tags from the [Anima](https://civitai.com/models/118825/anima-pencil-xl) model, built by [Mooshieblob](https://github.com/Mooshieblob1).

Browse, search, and preview thousands of artist styles. Right-click any card to instantly copy the tag for use in your prompts.

## Features

- Browse all artists included in the Anima model with preview images
- Search by artist tag with live suggestions
- Sort by post count, name, or "uniqueness" (hidden gem scoring)
- Adjustable card size with a logarithmic slider
- Light / Dark / Auto theme
- Click an artist name in the lightbox to open their Danbooru page
- Free forever

## Want to generate AI art with these styles?

This gallery pairs perfectly with **[MooshieUI](https://github.com/Mooshieblob1/MooshieUI)** — a beginner-friendly front-end for ComfyUI that makes generating AI images straightforward, no technical setup required.
